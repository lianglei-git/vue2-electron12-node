# vue2 + electron12 + Node 

此项目基于vue3.0cli electron12.x node做的架子
也只是作为简单测试和学习的同学来用， 有需要自取。

这个项目很是个自己做点小应用， 后续我会发出来我自己做的项目
## Project setup
```
yarn || yarn install
```

### Compiles and hot-reloads for development
```
yarn electron:serve
```

### Compiles and minifies for production
```
yarn electron:build
```
